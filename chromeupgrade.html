Updating Foxtrick in Chrome

Important: This is the procedure for updating a 0.18.x.x (NG) version of Foxtrick. If you are upgrading from 0.17.x.x you must follow the instructions here.

Video guide:

Text instructions:

  1. Download the Foxtrick for Chrome zip file.
  2. Extract the contents of the zip file into the folder where you previously installed Foxtrick, overwriting your previous install.
  3. Open 'Manage Extensions' in Chrome and click on the Reload icon.

If you were already logged in to hattrick while you did this, refresh the page.

Your browser may display messages like the following saying that support for the extension will be ending soon: This is expected behaviour, but Foxtrick should still operate correctly for now.
