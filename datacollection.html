Data Collection Policy

This document provides detailed information on the data Foxtrick collects and how it is handled for maintenance and debugging.

1. Anonymised Analytics

To help us monitor the stability of the extension, we collect basic statistics for each session (on page load):

  • Data transmitted: The Foxtrick version number and a simple indicator of whether a script error has occurred.
  • Privacy: While your Manager ID is technically transmitted as part of this request, it is not retained and cannot be accessed by the Foxtrick developers. No personal identifiers or account details are stored during this process.

2. Error Reporting and Debugging

Foxtrick uses a two-tiered system for handling technical issues:

A. Automated Error Reporting
When a critical error occurs, the extension sends diagnostic data to a third-party logging service. This includes:

  • Hattrick Context: Manager ID and Name, Team ID and Name, Teamâ€™s Country, the current Hattrick page URL, and the referrer URL.
  • Hattrick Settings: Theme, language, text direction, time zone, date format, and currency.
  • Technical Information: Error message and stack trace, browser name and version, and Operating System type/version.

B. Manual Error Reporting
If you choose to send a manual report, the following additional data is included to provide deeper context:

  • Foxtrick Preferences: Your extension settings. We automatically strip sensitive tokens and personal data from these settings, including Foxtrick CHPP connection tokens, saved message templates, and saved image/sound data.
  • Debug Log: A technical log of the extension's recent activity. Note: We do not actively strip data from this log. While it is primarily composed of diagnostic information relating to Foxtrick and its interaction with Hattrick, it may contain any data processed by the extension during your session (such as details of pages viewed or game IDs).

3. Data Retention and User Rights

  • Retention: All collected diagnostic and error data is automatically deleted after 90 days.
  • Right to Deletion: All stored data pertaining to a specific user can be deleted upon request. Please contact the developers via Hattrick Mail to initiate this.
  • Transmission: Data is transmitted securely via the fetch API over HTTPS.
  • Usage: Data is used strictly for technical debugging. We do not sell or share this information.

4. Your Control

You can manage these features at any time:

  1. Open Foxtrick Preferences.
  2. Navigate to Categories -> Core.
  3. Adjust the settings for the Reporter module.
