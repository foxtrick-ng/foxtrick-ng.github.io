Foxtrick

Browser extension for the Hattrick online football manager game

Chrome - install in place

  1. Export and save your current Foxtrick settings.
  2. Download Chrome zip file for local install.
  3. Make a folder to store the extension in, and extract the contents of the downloaded zip file there.
  4. Open 'Manage Extensions' in chrome and disable/delete your current version of Foxtrick if you have one installed.
  5. Turn on Developer Mode by clicking the toggle at the top right.
  6. Click 'Load Unpacked' at the top left and select the folder where you extracted the zip file contents.

The extension should now be installed. Import your Foxtrick settings if necessary.

If you click on the Details button for the extension, you can enable 'Allow in incognito' and 'Pin to toolbar' if you require these things.
