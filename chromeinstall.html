Installing Foxtrick in Chrome

Important When installing version 0.19.0.0 onwards, steps 4 and 5 in the instructions below are no longer required. You may safely skip these steps.

Video guide:

Text instructions:

Important Do not delete the folder where you have installed Foxtrick after the install is complete.

  1. Export and save your current Foxtrick settings if possible.
  2. Download the Foxtrick for Chrome zip file.
  3. In a suitable location, make a folder to store the extension in and extract the contents of the downloaded zip file there.
  4. In Chrome, paste the following into the URL field: chrome://flags/#allow-legacy-mv2-extensions You can skip this step for version 0.19 onwards.
  5. Make sure the highlighted option is Enabled Like this. Then click 'Relaunch' at the bottom right. You can skip this step for version 0.19 onwards.
  6. Open 'Manage Extensions' in Chrome and disable your current version of Foxtrick if you have one installed.
  7. Turn on Developer Mode by clicking the toggle at the top right.
  8. Click 'Load Unpacked' at the top left and select the folder where you extracted the zip file contents.
  9. The extension should now be installed. Import your Foxtrick settings if necessary.

If you click on the Details button for the extension, you can enable 'Allow in incognito' and 'Pin to toolbar' if you require these things.

In 0.18 versions your browser may display messages like the following saying that support for the extension will be ending soon: This is expected behaviour, but Foxtrick should still operate correctly for now.
