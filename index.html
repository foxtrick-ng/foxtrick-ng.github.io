Foxtrick

Download the latest version of the Foxtrick browser extension for Hattrick.

Firefox

Download foxtrick-0.18.5.0 for Firefox

If you are migrating from the old version of Foxtrick, ie 0.17.9 (beta), please go to the Foxtrick options and export your preferences. These can be imported into the new version after you have installed it.

Make sure you do not have more than one version of Foxtrick active at the same time. You can disable older versions in the Extension Manager and then delete them once you have transferred your preferences and are happy with the new version.

Android users: See the video here for a guide to installing Foxtrick on Android.
