Foxtrick

Browser extension for the Hattrick online football manager game

Firefox

Download foxtrick-0.18.2.0 for Firefox

Make sure you do not have more than one version of Foxtrick active at the same time. You can disable older versions in the Extension manager and then delete them once you are happy you no longer need them.

Chrome

It is not currently possible for us to update Foxtrick on the Chrome extension store, but it is possible to install it locally using the process explained here

Please note: when installed this way Foxtrick will not automatically update. You will need to repeat the manual process above each time you want to update to a new Foxtrick version.
